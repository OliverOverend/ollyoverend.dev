# Gotchas

Here are some things that have caught me out in the past.

- Windows has a maximum path length limit of 260 characters. See [https://learn.microsoft.com/en-us/windows/win32/fileio/maximum-file-path-limitation?tabs=registry](https://learn.microsoft.com/en-us/windows/win32/fileio/maximum-file-path-limitation?tabs=registry).
- Overflow errors. See [https://numpy.org/doc/stable/user/basics.types.html#overflow-errors](https://numpy.org/doc/stable/user/basics.types.html#overflow-errors).
